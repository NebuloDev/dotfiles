# KDE Terminals

Terminals are a surprisingly convoluted aspect of any Linux desktop.

Personally, I'm a fan of slide-in terminals, opposed to dedicated windows.

Here is a list of my personal go-to's:

- Konsole - KDE's default terminal, a good to have [[Arch](https://archlinux.org/packages/extra/x86_64/konsole/), [CachyOS](https://packages.cachyos.org/package/cachyos-extra-v3/x86_64_v3/konsole)]
- Yakuake - The slide-in version of Konsole, uses same profiles [[Arch](https://archlinux.org/packages/extra/x86_64/yakuake/), [CachyOS](https://packages.cachyos.org/package/cachyos-extra-v3/x86_64_v3/yakuake)]
- Ghostty - The most customisable one, possibly less performant [[Arch](https://archlinux.org/packages/extra/x86_64/ghostty/), [CachyOS](https://packages.cachyos.org/package/cachyos-extra-v3/x86_64_v3/ghostty)]
    - As of v1.2, supports slide-in terminals

## Configuration

### Yakuake

In the yakuake settings:

- Disabled "Show title bar" 
- Disabled "Open window after program start"
- Enabled "Show system tray icon" (For settings access)
- Theme: [Qogir Materia Dark](https://store.kde.org/p/2230336) (Available in KDE Store)
